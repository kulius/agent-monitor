# Sound Effects

Place the following sound files in this directory:

- `typing.mp3` - Keyboard typing sound (loopable, ~1-2 seconds)
- `help.mp3` - Alert/notification sound for help requests (~0.5-1 second)
- `complete.mp3` - Success/completion chime (~1-2 seconds)

## Recommended Sources (Free)

1. **freesound.org** - Large library of free sounds
2. **mixkit.co** - Free sound effects
3. **sfxr.me** - Generate 8-bit style sounds online
4. **pixabay.com/sound-effects** - Free sound effects

## Tips

- Keep files small (<100KB each) for fast loading
- Use MP3 format for best browser compatibility
- For pixel art style, consider 8-bit/chiptune sounds from sfxr.me
